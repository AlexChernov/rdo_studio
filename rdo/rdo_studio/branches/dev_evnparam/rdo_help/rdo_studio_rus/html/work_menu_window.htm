МЕНЮ ОКНА

Данное меню предоставляет функции для работы с MDI-окнами.

Пункт меню Комбинация клавиш Функция
Расположить каскадом   Располагает каскадом все открытые MDI-окна
Расположить заголовками   Располагает все открытые MDI-окна равномерно по всей доступной области
Упорядочить свернутые   Упорядочивает свернутые MDI-окна
Имена окон   Под чертой, отделяющие первые три пункта меню, находятся имена открытых MDI-окон. Окно становится активным, если выбрать его имя.
