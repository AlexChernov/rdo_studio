Qt - кросс-платформенный инструментарий разработки ПО на языке программирования C++.

1. Установить Qt5 можно тремя различными способами. Я предлагаю пользовать (1)

1.1. install.qt.from.installer.txt
Через инсталятор. Самый простой вариант, он требует наличие Visual Studio 2012.

1.2. install.qt.from.sources.txt
Через компиляцию исходников. Самый сложный и долгий, но надёжный imho. Можно использовать разные версии Visual Studio, в том числе и 2008.

1.3. install.qt.from.binary.txt
Через заранее скомпилированные бинарники. Это компромисный вариант, который подходит *только* для Visual Studio 2008 + SP1, а времени занимает гораздо меньше, чем (2)

2. Дополнение для Visual Studio
Чтобы во время отладки видеть значения объектов Qt в удобном виде устанавите Visual Studio Add-in for Qt5 (http://qt-project.org/downloads)
На Visual Studio 2012 Express for Desktop поставить нельзя.
