PostgreSQL - ядро базы данных, с которым работает РДО.


1. Загрузка инсталятора
http://www.enterprisedb.com/products-services-training/pgdownload

Находим и загружаем последнюю стабильную версию PostgreSQL для своей платформы. На текущий момент это
Version 9.2.2


2. Установка
Оставляем все предложеные параметры со значениями по умолчанию, кроме одного: пароль к учетной записи "rdo"

3. Настройка окружения и IDE
3.1. Добавляем переменную окружения PSQL

Находим директорию, куда поставили PostgreSQL, её содержимое выглядит как:
bin\
data\
doc\
include\
и т.д.

И прописываем PSQL, например
PSQL=C:\Program Files\PostgreSQL\9.2

в Win 7/8 это делается так:
Панель управления -> Учетные записи пользователей -> Изменение переменных среды -> создаём QTDIR и прописываем значение

Не забывам перелогиниться в винде или заново открыть Visual Studio
