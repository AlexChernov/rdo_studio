Инструкция установки РДО из исходников

1. Перед прочтением. Команда вида
QTDIR=D:\Qt\Qt5.0.1\5.0.1\msvc2010
означает, что надо задать переменную окружения.

1.1 Для Windows 7 это делается так:
Панель управления -> Учетные записи пользователей -> Изменение переменных среды -> создаём QTDIR и прописываем значение.
После этого лечше перелогиниться (не путать с перезагрузкой) или перезапустить программу, которой была нужна переменная.

1.2 Для Linux
export QTDIR=/home/username/Qt5.2.1/5.2.1/gcc/
Добавить подобную строчку в .bashrc (или .profile под Убунтой)
После этого лечше перелогиниться (не путать с перезагрузкой) или перезапустить программу, которой была нужна переменная.

2. Устанавливаем 
install.boost.txt
install.qt.txt
install.cmake.txt
install.svn.txt

3. Компилируем
make.project.txt
