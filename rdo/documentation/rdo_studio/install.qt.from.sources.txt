0. Найти на винте 7.5Gb и несколько часов терпения

1. Загрузка Qt
http://qt-project.org/downloads

Ищем ссылку в виде:
The source code is available as a zip (261 MB)

Архив должен называться
qt-everywhere-opensource-src-<версия>.zip

Качаем, распаковываем в любое место, например, в C:\qt-4.8.1. Эта директория должна содержать
bin
demos
doc
examples
imports
include
lib
и др. файлы и папки

2. Конфигурирование Qt
2.1 Для конфигурирования надо установить Perl
http://www.activestate.com/activeperl/downloads
После установки необходимо перелогиниться в винде (перезагружаться не обязательно)

2.2 Запуск конфигуратора (команды можно сразу выполнять из консоли, которую придётся открыть в пункте 3.1). Запускать кофигуратор именно из его директории. Если выдает Error, то попробуйте запустить консоль от имени Администратора. Длительность - минут 10.
cd C:\qt-4.8.1
configure.exe -debug-and-release -opensource

3. Компиляция Qt
3.1 Компиляция производится из консоли MS Visual Studio
Пуск\Microsoft Visual Studio 2008\Visual Studio Tools\Visual Studio 2008 Command Prompt
или
Пуск\Поиск\Visual Studio 2008 Command Prompt

Длительность компиляции может достигать нескольких часов. Скомпиляровать под виндой можно двумя способами: либо 3.2, либо 3.3
3.2 Использовать nmake
cd C:\qt-4.8.1
nmake

3.3 Использовать jom. Работает быстрее на многоядерных машинах, примерно на полчаса быстрее получается.
3.3.1 Для этого надо скачать .exe
http://qt-project.org/wiki/jom

3.3.2 Нужно распаковать в любую диреторию. Например:
c:\jom\

3.3.3 Прописать в переменную среды PATH путь из 3.3.2
в Win 7 это делается так:
Панель управления -> Учетные записи пользователей -> Изменение переменных среды -> ищем Path и дописываем
Если Path нет, то заводим.

теперь можно вместо 3.2 выполнить:
cd C:\qt-4.8.1
jom

4. Настройка окружения и IDE

4.1. Добавляем переменную окружения
QTDIR=C:\qt-4.8.1
Не забывам перелогиниться в винде.

4.2. Для Windows запускаем Visual Studio, заходим в настройки
Tools/Options/Projects and Solutions/VC++ Directories

Platform             = Win32
Show directories for = Include files, добавляем $(QTDIR)\include
Show directories for = Library files, добавляем $(QTDIR)\lib
