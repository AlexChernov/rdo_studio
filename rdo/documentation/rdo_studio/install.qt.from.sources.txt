0.1 На хабре тоже есть инструкция для компиляции Qt5 по 2008 студию: http://habrahabr.ru/post/164357/

1. Загрузка Qt
http://qt-project.org/downloads

Ищем ссылку в виде:
The source code is available as a zip (270 MB)

Архив должен называться
qt-everywhere-opensource-src-<версия>.zip

Качаем, распаковываем в любое место, например, в C:\qt-5.0.0. Эта директория должна содержать
gnuwin32
qtactiveqt
qtbase
qtdeclarative
qtdoc
qtgraphicaleffects
qtimageformats
qtjsbackend
qtmultimedia
и др. файлы и папки

2. Конфигурирование Qt
2.1 Для конфигурирования надо установить ActiveState Perl
http://www.activestate.com/activeperl/downloads
После установки необходимо перелогиниться в винде (перезагружаться не обязательно)

2.2 Еще надо установить Python
http://www.python.org/download/releases/2.7.2/
И прописать путь к нему его в PATH. Можно в консоли из 3.1 набрать
set PATH=C:\Program Files (x86)\Python27;%PATH%

2.3 И библиотеку ICU с cygwin'ом. См. install.icu.txt

2.4 Запуск конфигуратора (команды можно сразу выполнять из консоли, которую придётся открыть в пункте 3.1). Запускать кофигуратор именно из его директории. Если выдает Error, то попробуйте запустить консоль от имени Администратора. Длительность - минут 10.
cd C:\qt-5.0.0
configure.bat -debug-and-release -opensource -opengl desktop -icu

3. Компиляция Qt
Компиляция производится из консоли MS Visual Studio
Пуск\Microsoft Visual Studio 2008\Visual Studio Tools\Visual Studio 2008 Command Prompt
или
Пуск\Поиск\Visual Studio 2008 Command Prompt

Далее
cd C:\qt-5.0.0
nmake

4. Настройка окружения и IDE

4.1. Добавляем переменную окружения
QTDIR=C:\qt-5.0.0
Не забывам перелогиниться в винде.

4.2. Для Windows запускаем Visual Studio, заходим в настройки
Tools/Options/Projects and Solutions/VC++ Directories

Platform             = Win32
Show directories for = Include files, добавляем $(QTDIR)\qtbase\include
Show directories for = Library files, добавляем $(QTDIR)\qtbase\lib
