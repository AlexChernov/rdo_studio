Библиотека ICU является сторонней для Qt. Используется для работы с юникодом WebKit'а. А без последнего плохо отображаются страницы документации РДО.

0. Можно скачать скомпилированные бинарники с google-docs
https://docs.google.com/folder/d/0B-tnm4ER5l74c0hnRFl4LTVHbDA/edit?usp=sharing
Файл
qt-5.1.1/icu-51.2-bin.7z

Но собраны они под Visual Studio 2008 SP1. Если у вас другая версия компилятора, но этот вариант не подходит.

1. Скачать исходники последней версии. На текущий момент - это icu4c-51_2-src.zip
http://site.icu-project.org/

2. Разархивировать, например, в D:\Downloads\qt\icu

2. Для сборки надо поставить cygwin. См. install.cygwin.txt
3. Теперь консольный квест
3.1 Открываем спец. консоль *с правами администратора*
Пуск\Поиск\Visual Studio 2008 Command Prompt

3.3 Добавляем cygwin в PATH прямо из консоли
set PATH=D:\cygwin64\bin;%PATH%

3.4 Запускаем консоль cygwin'а
D:\cygwin64\bin\bash.exe

4. Конфигурирование и компиляция
cd /cygdrive/d/Downloads/qt/icu/source/
./runConfigureICU Cygwin/MSVC

Если ошибка, то выходим из cygwin'а и запускаем
"C:\Program Files (x86)\Microsoft Visual Studio 9.0\VC\bin\vcvars32.bat"
Повторяем 3.4. Если удачно, то

make
make install

5. После сборки, можно удалить D:\Downloads\qt\icu, т.к. всё нужное будет в D:\cygwin64\usr\local\

6. Для доступа к скомпилённым dll нужно скопировать
icudt51.dll
icuin51.dll
icuio51.dll
icule51.dll
iculx51.dll
icutest51.dll
icutu51.dll
icuuc51.dll

icudt.lib
icuin.lib
icuio.lib
icule.lib
iculx.lib
icutest.lib
icutu.lib
icuuc.lib

в
%(QTDIR)/bin
, например, C:\qt-5.1.1\qtbase\bin

Именно отсюда их будет брать РДО при компиляции

Сами dll лежат или в D:\cygwin64\usr\local\lib или выкачиваются с google-disk'а, см. пункт 0
