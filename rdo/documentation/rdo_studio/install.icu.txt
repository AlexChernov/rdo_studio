Библиотека ICU является сторонней для Qt. Используется для работы с юникодом WebKit'а. А без последнего плохо отображаются страницы документации РДО.

0. Можно скачать скомпилированные бинарники с google-docs
https://docs.google.com/folder/d/0B-tnm4ER5l74c0hnRFl4LTVHbDA/edit?usp=sharing
Файл
qt-5.1.1-vs2008sp1/icu-51.2-bin.7z
или
qt-5.1.1-vs2013/icu-52.1-bin.7z
И сразу к шагу 6

Если у вас другая версия компилятора, но этот вариант не подходит.

1. Скачать исходники последней версии. На текущий момент - это icu4c-52_1-src.zip
http://site.icu-project.org/

2. Разархивировать, например, в c:\rdo\icu

2. Для сборки надо поставить cygwin. См. install.cygwin.txt
3. Теперь консольный квест
3.1 Открываем 'правильную консоль' [install.txt:2.1] *с правами администратора*
Пуск\Поиск\Visual Studio 2008 Command Prompt

3.3 Добавляем cygwin в PATH прямо из консоли
set PATH=C:\Program\cygwin\bin;%PATH%

3.4 Запускаем консоль cygwin'а
bash.exe

4. Конфигурирование и компиляция
cd /cygdrive/c/rdo/icu/source/
./runConfigureICU Cygwin/MSVC

Если ошибка, то выходим из cygwin'а
exit
и запускаем [install.txt:2.2]

Повторяем 3.4. Если удачно ("ICU for C/C++ 52.1 is ready to be built."), то

make
make install

5. После сборки можно удалить c:\rdo\icu, т.к. всё нужное будет в C:\Program\cygwin\usr\local\

#6. Для доступа к скомпилённым dll нужно скопировать
#icu*.dll
#icu*.lib
#
#в
#%(QTDIR)/bin
#, например, C:\qt-5.1.1\qtbase\bin
