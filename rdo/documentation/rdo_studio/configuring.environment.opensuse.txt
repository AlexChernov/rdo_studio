Руководство по настройке среды для сборки компиляции в операционной системе GNU/Linux openSUSE
Должно также в точности подходить для всех ОС на ядре Linux, использующих систему управления пакетами zypper, при наличии в их репозиториях соответствующих пакетов.

0. Для сборки rdo нам понадобятся следующие инструменты:
 - boost
 - cmake
 - make
 - bison
 - flex
 - gcc

1. Установка ОС:
Сборка РДО тестировалась в операционной системе openSUSE 12.2.

2. Установка инструментария:
2.1. Ленивая установка:
Заходим в YAST-> управление программные обеспечением и выбираем все необходимые пакеты из пункта 0 с учетом примечаний пункта 1.

2.2. Из консоли
с помощью zypper устанавливает все необходимые пакеты (возможны отличия имен, вариант не пробовался)

3. Enjoy
