Qt - кросс-платформенный инструментарий разработки ПО на языке программирования C++.

В связи с бурным развитием кросс-платформенных начинаний в РДО, GUI системы активно портируется на Qt.

0. Найти на винте до 1.2Gb.
 + УСТАНОВЛЕННАЯ MSVC.

1. Загрузка Qt SDK
http://qt-project.org/downloads

Находим и загружаем последнюю стабильную версию Qt. На текущий момент это
Qt libraries 4.8.3 for Windows (VS 2008, 234 MB)

Примечания: для загрузки нужен хороший интернет и в качестве браузера лучше использовать opera с докачной, в chrome закачка часто обрывается на 80-90%

2. Установка
Просто запустить только что скаченный инсталлятор

3. Настройка окружения и IDE
3.1. Добавляем переменную окружения QTDIR

Находим директорию, куда поставили Qt, её содержимое выглядит как:
bin\
imports\
include\
lib\
и т.д.

И прописываем QTDIR, например
QTDIR=C:\Qt\4.8.3

в Win 7 это делается так:
Панель управления -> Учетные записи пользователей -> Изменение переменных среды -> создаём QTDIR и прописываем значение

Не забывам перелогиниться в винде или заново открыть Visual Studio

3.2. Для Windows запускаем Visual Studio, заходим в настройки
Tools/Options/Projects and Solutions/VC++ Directories

Platform             = Win32
Show directories for = Include files, добавляем $(QTDIR)\include
Show directories for = Library files, добавляем $(QTDIR)\lib
