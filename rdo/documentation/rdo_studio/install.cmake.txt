CMake - the cross-platform, open-source build system

После переноса РДО на POSIX-совместимые ОС CMake должна стать основным инструментом генерации файлов управления сборкой системы.

1. Для Windows
1.1. Скачать и установить CMake
    http://www.cmake.org/
    текущая версия 2.8.11

1.2 Путь к cmake должен быть в PATH
set PATH=D:\Program Files\CMake 2.8\bin;%PATH%

2. Для GNU/Linux
    Установить из репозиториев своего дистрибутива, например, для Debian/Ubuntu:

apt-get install cmake
