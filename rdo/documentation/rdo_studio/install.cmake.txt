CMake - the cross-platform, open-source build system

    После переноса РДО на POSIX-совместимые ОС CMake должна стать основным инструментом генерации файлов управления сборкой системы.

1. Для Windows
1.1. Скачать CMake
    http://www.cmake.org/
    текущая версия 2.8.5

1.2. Установка
1.2.1. Запустить инсталятор и следовать его указаниям

1.3. Настройка окружения для BOOST
    Необходимо добавить три переменные окружения:

    Переменная                      Как она выглядит в моем случае
    
    BOOST_ROOT              -       C:\boost\boost_1_45_0_msvc_8

    Внутри BOOST_ROOT должны находиться две папки :

    1) include, а в ней boost со всеми его заголовочными файлами
    2) lib, а в ней все .lib

    Чтобы CMAKE обнаружил bison и flex необходимо в PATH добавить путь к этим исполняемым файлам.

    Путь к cmake тоже должен быть в PATH, как минимум для работы MinGW.

1.4. Настройка окружения для Bison и Flex
    В PATH необходимо добавить путь к bison.exe и flex.exe
    В моем случае D:\repository\rdo_studio\thirdparty\bison_flex\bin


2. Для GNU/Linux
    Установить из репозиториев своего дистрибутива.
    Например, для Debian:
    apt-get install cmake
