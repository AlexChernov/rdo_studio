Руководство по настройке среды для сборки РДО в операционной системе GNU Debian/Linux.
Должно также в точности подходить для всех ОС на ядре Linux, использующих систему управления пакетами Advanced Packaging Tool, при наличии в их репозиториях соответствующих пакетов.

0. Для работы над проектом понадобятся следующие инструменты:
0.1. Для доступа к репозиторию:
 - subversion

0.2. Для компиляции rdo:
 - boost
 - cmake
 - make
 - bison
 - flex
 - gcc

0.3. Для автоматического создания документации:
 - doxygen
 - dot (входит в пакет graphviz)

1. Установка ОС:
Wheezy версия debian'а была выбрана из-за boost'а 1.46, которого на сегодняшний день нет в стабильной ветке репозитория.
Поэтому если перейти на Wheezy будет возможна "ленивая" версия установки всех инструментов, представленная ниже.
С другой стороны, boost можно скачать с официального сайта и собрать самостоятельно на любом дистрибутиве (например Squeeze).

2. Установка полного набора инструментов:
$ apt-get install libboost-all-dev bison flex cmake make gcc subversion doxygen graphviz

3. Enjoy
