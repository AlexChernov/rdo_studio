Настрока Яндекс Диска на debian/ubuntu linux server

1) Установить davfs2:

    sudo apt-get install davfs2

2) Обновить /etc/fstab ( добавить строку  ) :

    https://webdav.yandex.ru                  /webdav         davfs   noauto,rw         0       0

3) Обновить /etc/davfs2/secrets ( добавить строчку ) :

    https://webdav.yandex.ru "login" "password"

4) Ручное монтирование :

    sudo mount /webdav

5) Ручное размонтирование :

    sudo umount /webdav

6) Автоматическое монтирование ( добавить строку в /etc/rc.local ) :

    mount /webdav
