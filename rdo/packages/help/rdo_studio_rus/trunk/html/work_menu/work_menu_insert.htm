МЕНЮ ВСТАВКА

Данное меню предоставляет функции для вставки синтаксических конструкций языка и заготовок (шаблонов) для написания элементов модели.

Пункт меню Комбинация клавиш Кнопка Функция
RTP, RSS, EVN, PAT, DPT, PRC, FRM, FUN, SMR, PMD   При выборе данных пунктов меню открывается подменю, содержащее синтаксические конструкции и шаблоны для соответствующего объекта модели. Выбранный из подменю элемент вставляется в текст модели
Функции   При выборе данного пункта меню открывается подменю, содержащее перечень стандартных арифметических и логических функций языка РДО. При выборе функции из подменю в текст модели вставляется имя выбранной функции
