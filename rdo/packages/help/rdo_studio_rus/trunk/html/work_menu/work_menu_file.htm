МЕНЮ ФАЙЛ

Данное меню предоставляет функции для работы с файлом модели.

Пункт меню Комбинация клавиш Кнопка Функция
Новая модель Ctrl+N Создание новой модели
Открыть... Ctrl+O Открытие существующей модели
Недавние   Повторное открытие ранее открывавшейся модели
Закрыть   Закрывает текущую модель
Сохранить Ctrl+S Запись изменений в модели
Сохранить как...   Запись модели под другим именем
Сохранить всё Запись всех открытых документов
Выход Alt+F4   Выход из программного комплекса. Если модель содержит не сохраненные изменения, то будет предложено их сохранить.
