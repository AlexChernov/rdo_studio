СОХРАНЕНИЕ МОДЕЛИ

Сохранить модель можно путем выбора пункта меню Файл/Сохранить, нажав комбинацию клавиш Ctrl+S или с помощью соответствующей кнопки на панели инструментов проекта - . Команда Файл/Сохранить всё производит полностью аналогичные действия.

Существует возможность сохранения модели под другим именем. Для этого необходимо выбрать пункт меню Файл/Сохранить как. При этом будет выдано стандартное диалоговое окно WINDOWS для сохранения файла, где пользователю предлагается ввести новое имя файлов модели (файла прогона с расширением *.smr):

При записи модели под другим именем фактически записывается с новым именем только файл (объект) прогона (*.smr). Чтобы изменить имена других файлов модели, необходимо:

  • Для файлов с расширением *.pat, *.rss, *.rtp, *.opr, *.frm, *.fun, *.dpt и *.pmd указать новые значения в объекте прогона на закладке SMR (см. подробнее) и записать модель командой Файл/Сохранить. Если в именах файлов будут допущены ошибки, то может возникнуть ошибка открытия модели.
  • Для файлов с расширением *.pmv (результаты моделирования) и *.trc (трассировка) так же можно указать новые значения в объекте прогона на закладке SMR (см. подробнее) параметрам Results_file и Trace_file соответственно, но сформированными под новыми именами они будут после очередного прогона, а не в момент записи модели.
